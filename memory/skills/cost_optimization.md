## 省钱原则

1. **MVP 思维**：先用免费方案验证，有流量再升级
2. **善用免费额度**：GitHub Pages / Vercel 免费版足够个人站点
3. **延迟消费**：域名等有明确需求再买
4. **资源优化**：纯 HTML/CSS 无需服务器渲染，成本最低
5. **图片优化**：用 CSS/SVG 代替图片，减少存储和带宽

## 推荐路径

```
阶段一（当前）：预览验证 → ¥0
    ↓
阶段二：GitHub Pages 上线 → ¥0
    ↓
阶段三：绑定自定义域名 → ¥55/年
    ↓
阶段四：如需国内高速 → Vercel/腾讯云
```

## 注意事项

- 免费方案可能在国内访问较慢，影响用户体验
- 如果面向国内客户，建议至少用 Vercel
- 域名备案是国内服务器的前置条件（约 20 天）
