## 六、前店后院方案详情

### 什么是"前店后院"

| 区域 | 定义 | 典型场景 |
|------|------|----------|
| **前店** | 美妆零售区 | 产品展示、开架试用、快速收银 |
| **后院** | 美容服务区 | 皮肤护理、面部SPA、体验式服务 |

### 六大模块

| 模块 | 功能描述 |
|------|----------|
| **会员一体化** | 统一会员档案、积分通用、权益互通 |
| **库存打通** | 前后院库存共享、实时同步、预警提醒 |
| **绩效联动** | 零售+服务业绩自动核算、员工提成清晰 |
| **预约-服务-销售闭环** | 在线预约→到店核销→服务完成→零售转化 |
| **数据化运营** | 实时经营看板、客流分析、转化漏斗 |
| **全渠道接入** | PC+APP+小程序，多端协同 |

### 价值数据

| 指标 | 提升幅度 |
|------|----------|
| 客户停留时长 | 提升3倍 |
| 会员复购率 | 提升50%+ |
| 连带购买率 | 提升40%+ |
| 客单价提升空间 | 2倍 |

### 嘉悦国际案例详情

**客户背景：**
- 成立20年，港资企业
- 代理12大国际品牌（Natura Bissé、[comfort zone]等）
- 1000余家五星级酒店会所和美容会所
- 所有门店均为直营店
- 业务模式：前店后院

**解决的问题：**
1. 会员统一：前后院共享会员池，权益通用
2. 订单统一：零售+服务混合订单，自动拆分
3. 库存统一：前后院库存实时同步
4. 业绩统一：零售和服务业绩自动归属
5. 数据统一：总部实时看到全店经营数据

## 九、核心卖点总结

1. **"前店后院"一体化** — 美业专属，解决零售+服务混合业态痛点
2. **9年美业沉淀** — 50+美业客户经验，懂美业

5. **AI能力加持** — 快速交付，创新业务快速落地
